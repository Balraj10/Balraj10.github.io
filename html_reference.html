HTML References


20 Examples of HTML Tags

<!DOCTYPE html>

!DOCTYPE html declaration is used to inform a website visitor's browser that the document being rendered is an HTML document.

<head>

A container for all the head elements.

<h1>

Used to define HTML headings.

<title>

Required in all HTML documents and it defines the title of the document.

<body>

Defines the document's body

<a>

Defines a hyperlink, which is used to link from one page to another.

<hr>

Defines a thematic break in an HTML page

<style>

Used to define style information for an HTML document.

<div>

Defines a division or a section in an HTML document.

<table>

Defines an HTML table.

<center>

Used to center-align text.

<p>

Defines a paragraph.

<br>

inserts a single line break.

<img>

Defines an image in an HTML page.

<video>

Specifies video, such as a movie clip or other video streams.

<iframe>

Specifies an inline frame.

<header>

Represents a container for introductory content or a set of navigational links.

<footer>

Defines a footer for a document or section.

<ul>

Defines an unordered (bulleted) list.

<li>

Defines a list item.
